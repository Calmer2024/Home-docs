# JetBrains IDE 核心快捷键文档

这是一份为您准备的JetBrains IDE（如IntelliJ IDEA, PyCharm, WebStorm, GoLand, CLion等）常用快捷键；

JetBrains系列IDE的快捷键在不同操作系统上略有差异，本文档将同时提供Windows/Linux和macOS的快捷键。

### **JetBrains IDE 核心快捷键文档**

### **使用技巧**

- **忘记快捷键怎么办？** 使用 `查找操作 (Find Action)` (`Ctrl + Shift + A` / `Cmd + Shift + A`)，直接输入你想要做的事情，比如 "format code"，IDE会告诉你对应的操作和快捷键。
- **Keymap自定义**: 你可以在 `Settings/Preferences` -> `Keymap` 中查看、修改甚至导入你习惯的快捷键方案（例如Eclipse或VS Code的键位）。
- **双击`Shift`大法**: 这是最强大、最常用的快捷键。它可以搜索任何东西：类、文件、符号、操作、设置项等。当你不知道该用什么功能时，就双击`Shift`。
